Networking Class

About Class

A networking class is a class that learns about installing or troubleshooting a medium-sized enterprise network according to
the needs of the international industry. In this class, all children will be taught from the start (what is a network) so they can
get international certification so that children can compete when they are involved in the world of work later.
This class is oriented according to the curriculum created by renowned networking companies. including Cisco, Mikrotik, Ubiquiti, Aruba Network, Juniper, etc.

Advantages

One of the advantages of this class is the use of an international curriculum. so that children can take the international certification exam which can be used to get a job later.

Certification Program

in networking, classes study and have several international certifications that can be used to get a job later.
including:
Cisco: CCNA and CCNP
Mikrotik: MTCNA, MTCRE, MTCINE
Ubiquiti: UEWA and UBWA
Aruba Network: ACMA and ACMP
Juniper: JNCIA and JNCIP
etc.

Programming Class

About Class

Programming Class is a class that learns about how to learn programming languages, with materials that are always
updated according to the times both in terms of source code and in terms of tools/libraries.
In this class, all children will be taught from the start (what is a programming ) so they can get international
certification so that children can compete when they are involved in the world of work later. In addition to learning programming languages, students will also be taught about web design and databases.
And this class uses a curriculum that is suitable for the job industry later.

Advantages

This class uses an international curriculum that has been designed by experts. we are oriented to several companies or educational communities. like dicoding, google, and apple academy. in this class, we learn many programming languages. including java, kotlin, flutter, swift, etc. So that children will continue to update and the skills are the same
as reliable developers.
Not only coding, in this class children, are also taught how to design an application. so that their application can attract the attention of others.

Certification Program

Programming class does not have many international certificates and only focuses on the portfolio. when children study
in this class they can take the international exam made by Google itself. namely Google Android Developer Associate.
and children can take the tests made by the programming communities. like dicoding.

Cloud Class

About Class

in the cloud class the children will learn to build private and public clouds, we will fully practice and fully build
their own cloud. And will be taught from the beginning to proficiently guided and taught by educators who
are very experienced in the world of cloud providers.

Advantages

Many large companies and startups use cloud services to support their infrastructure needs. So that this class
can be the most needed class in the world of work. and this class uses an international curriculum that is highly used by
the international industry. like AWS Cloud, Openstack, and Microsoft Azure.
